많은 기업에서 시도중인 프로세스 자동화, 효율성, 품질 향상의 목적으로 주로 사용되며 ai를 활용한 머신러닝은 기업의 운영 방식을 혁신하고 있습니다. 이러한 흐름 속에서 머신러닝(Machine Learning)은 핵심 기술로 자리 잡고 있습니다.
머신러닝은 데이터를 기반으로 패턴을 학습하고, 이를 통해 미래를 예측하거나 자동으로 의사결정을 수행하는 기술입니다. 
본 보고서에서는 머신러닝의 개념, 종류, 주요 알고리즘, 그리고 활용 사례를 중심으로 살펴보고자 합니다.

---

## 1. 머신러닝이란

머신러닝은 컴퓨터가 명시적인 규칙을 프로그래밍하지 않고, 데이터를 통해 스스로 학습하는 기술입니다.

` “데이터를 이용하여 모델을 학습시키고, 이를 통해 새로운 데이터를 예측하는 과정”으로 정의할 수 있습니다.`

---

## 2. 머신러닝의 종류

### 2.1 지도 학습 (Supervised Learning)

지도 학습은 정답(Label)이 있는 데이터를 기반으로 학습하는 방식입니다.

- 선형 회귀 (Linear Regression)  
- 로지스틱 회귀 (Logistic Regression)  
- 결정 트리 (Decision Tree)  
- 랜덤 포레스트 (Random Forest)  

---

### 2.2 비지도 학습 (Unsupervised Learning)

비지도 학습은 정답 없이 데이터의 구조나 패턴을 파악하는 방식입니다.

- K-Means 클러스터링  
- PCA (주성분 분석)  

---

### 2.3 강화 학습 (Reinforcement Learning)

강화 학습은 환경과 상호작용하면서 보상을 통해 학습하는 방식입니다.

---

## 3. 머신러닝 주요 알고리즘

- 선형 회귀: 연속적인 값을 예측하는 데 사용됩니다.  
- 로지스틱 회귀: 분류 문제를 해결하는 데 활용됩니다.  
- 결정 트리: 조건을 기반으로 데이터를 분할하는 모델입니다.  
- 랜덤 포레스트: 여러 개의 결정 트리를 결합한 앙상블 모델입니다.  

---

## 4. 머신러닝 학습 과정

머신러닝 프로젝트는 일반적으로 다음과 같은 단계로 진행됩니다.

1. 문제 정의  
2. 데이터 수집  
3. 데이터 전처리  
4. 모델 선택  
5. 모델 학습  
6. 모델 평가  
7. 결과 해석  

---

## 5. 간단한 구현 예시

```python
import numpy as np
from sklearn.linear_model import LinearRegression

X = np.array([[1], [2], [3], [4], [5]])
y = np.array([2, 4, 6, 8, 10])

model = LinearRegression()
model.fit(X, y)

prediction = model.predict([[6]])
print(prediction)
```
