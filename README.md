머신러닝은 데이터를 기반으로 패턴을 학습하고, 이를 통해 미래를 예측하거나 자동으로 의사결정을 수행하는 기술입니다. 
본 보고서에서는 머신러닝의 개념, 종류, 주요 알고리즘, 그리고 활용 사례를 중심으로 살펴보고자 합니다.

## 1. 머신러닝이란

머신러닝은 컴퓨터가 명시적인 규칙을 프로그래밍하지 않고, 데이터를 통해 스스로 학습하는 기술입니다.

` “데이터를 이용하여 모델을 학습시키고, 이를 통해 새로운 데이터를 예측하는 과정”으로 정의할 수 있습니다.`

아이가 수많은 동물 그림을 보고 “이건 강아지, 이건 고양이”를 구분하는 것과 비슷합니다. 
처음에는 헷갈리지만, 많이 경험할수록 구분 능력이 좋아집니다
즉 데이터 경험으로 성장하는 기계가 스스로 학습하는 기술입니다.

---

## 2. 머신러닝의 종류

### 2.1 지도 학습 (Supervised Learning)

지도 학습은 정답(Label)이 있는 데이터를 기반으로 학습하는 방식입니다.\

✔ 설명: 정답이 있는 데이터를 주고, 그 정답을 맞히도록 학습시키는 방법.\
✔ 예시: 과거 집값(데이터)과 실제 거래 가격(정답)을 학습 → 새로운 집의 예상 가격을 예측.\
✔ 비유: 선생님이 문제를 내고, 정답지를 보여주며 가르치는 수업.
- 선형 회귀 (Linear Regression)  
- 로지스틱 회귀 (Logistic Regression)  
- 결정 트리 (Decision Tree)  
- 랜덤 포레스트 (Random Forest)  

---

### 2.2 비지도 학습 (Unsupervised Learning)

비지도 학습은 정답 없이 데이터의 구조나 패턴을 파악하는 방식입니다.\

✔ 설명: 정답이 없는 데이터를 주고, 데이터 속 숨은 패턴을 찾는 방법.\
✔ 예시: 고객 데이터를 분석해 ‘비슷한 소비 패턴 그룹’으로 나누기.\
✔ 비유: 선생님이 정답은 안 알려주고, 학생들이 스스로 그룹을 만드는 활동.\
- K-Means 클러스터링  
- PCA (주성분 분석)  

---

### 2.3 강화 학습 (Reinforcement Learning)

강화 학습은 환경과 상호작용하면서 보상을 통해 학습하는 방식입니다.\

✔ 설명: 시행착오를 통해 보상을 최대화하는 방법.\
✔ 예시: 체스나 바둑 AI가 스스로 수를 두며 실력을 키움.\
✔ 비유: 아이가 게임을 하면서 점수를 높이기 위해 전략을 바꿔가는 과정.\

---

## 3. 머신러닝 주요 알고리즘

### 3.1 . 선형 회귀 (Linear Regression)

선형 회귀는 입력 변수와 출력 변수 간의 선형 관계를 가정하여 연속적인 값을 예측하는 알고리즘입니다.

데이터 간의 관계를 직선 형태로 표현합니다.

예를 들어, 공부 시간과 시험 점수 사이의 관계를 예측할 때 사용할 수 있습니다.

모델은 다음과 같은 형태로 표현됩니다.

y=ax+b

✔ 특징

구조가 단순하고 해석이 쉬움

빠르게 학습 가능

데이터가 선형 관계일 때 좋은 성능

✔ 사용 예시

집값 예측

매출 예측

---

### 3.2 . 로지스틱 회귀 (Logistic Regression)

로지스틱 회귀는 데이터를 특정 범주로 분류하는 데 사용되는 알고리즘입니다.

출력값이 0과 1 사이의 확률로 나타납니다.

일정 기준(보통 0.5)을 넘으면 특정 클래스에 속한다고 판단합니다.

시그모이드(Sigmoid) 함수를 사용합니다.

✔ 특징

이진 분류 문제에 적합

결과를 확률로 해석 가능

비교적 단순하면서도 강력한 모델

✔ 사용 예시

스팸 메일 분류

합격/불합격 예측

---

### 3.3 . 결정 트리 (Decision Tree)

결정 트리는 데이터를 여러 조건으로 나누어 트리 구조로 분류 또는 예측하는 알고리즘입니다.

"if-else" 형태로 의사결정을 수행합니다.

데이터를 특정 기준으로 계속 분할하면서 결과를 도출합니다.

✔ 특징

직관적이고 이해하기 쉬움

시각화 가능

과적합(Overfitting) 발생 가능

✔ 사용 예시

고객 이탈 예측

질병 진단

---

### 3.4 . 랜덤 포레스트 (Random Forest)

랜덤 포레스트는 여러 개의 결정 트리를 결합하여 더 안정적인 예측을 수행하는 앙상블 알고리즘입니다.

다양한 데이터 샘플과 변수로 여러 트리를 생성합니다.

각 트리의 결과를 종합하여 최종 결과를 도출합니다.

✔ 특징

과적합을 줄이고 성능 향상

노이즈에 강함

비교적 높은 예측 정확도

✔ 사용 예시

신용 평가

추천 시스템

---

## 4. 머신러닝 학습 과정

머신러닝 프로젝트는 일반적으로 다음과 같은 단계로 진행됩니다.

1. 문제 정의  
2. 데이터 수집  
3. 데이터 전처리  
4. 모델 선택  
5. 모델 학습  
6. 모델 평가  
7. 결과 해석  

머신러닝 학습 과정은 문제 정의부터 결과 해석까지 단계적으로 이루어지며, 각 단계가 서로 긴밀하게 연결되어 있습니다. \
문제를 명확히 정의하고, 적절한 데이터를 수집·전처리한 뒤, 알맞은 모델을 선택하여 학습과 평가를 수행해야만 의미 있는 결과를 얻을 수 있습니다. \
따라서 머신러닝은 단순히 알고리즘을 적용하는 기술이 아니라, 데이터를 기반으로 문제를 체계적으로 해결하는 종합적인 분석 과정이라고 볼 수 있습니다.

---

## 5. 간단한 구현 예시

```python
import numpy as np
from sklearn.linear_model import LinearRegression

X = np.array([[1], [2], [3], [4], [5]])
y = np.array([2, 4, 6, 8, 10])

model = LinearRegression()
model.fit(X, y)

prediction = model.predict([[6]])
print(prediction)
```

## 6. 머신러닝과 딥러닝

인공지능이 가장 큰 개념으로 머신러닝 개념을 포함합니다.\
인공지능(Artificial intelligence)은 인간의 학습 능력과 추론 능력, 지각 능력 등을 컴퓨터 프로그램으로 실현한 기술을 말하는데 그 연구 분야 중 하나가 바로 머신러닝입니다. \
딥러닝은 인공신경망을 이용한 머신러닝의 한 종류로 머신러닝의 하위 개념으로 볼 수 있습니다.

머신러닝과 딥러닝 모두 데이터를 분류하는 데 사용하는 기술이지만 두 기술의 가장 큰 차이점은 바로 사람의 개입 여부입니다.

머신러닝은 주어진 데이터를 인간이 먼저 처리합니다.\
사람이 먼저 컴퓨터에 특정 패턴을 추출하는 방법을 지시하고, 그 이후 컴퓨터가 스스로 데이터의 특징을 분석하고 축적합니다.\
예를 들어 사람이 먼저 개와 고양이의 사진을 보고 개와 고양이의 특징을 추출한 후 많은 예시를 통해 컴퓨터를 학습시키고 식별하게 만듭니다.\

반면에 딥러닝은 머신러닝에서 사람이 하던 패턴 추출 작업이 생략됩니다.\ 
컴퓨터가 스스로 데이터를 기반으로 학습할 수 있도록 정해진 신경망을 컴퓨터에게 주고, 어린아이가 학습하는 것처럼 경험 중심으로 학습을 수행합니다. \
즉, 인간이 개, 고양이의 특성을 추려 사전에 정의된 알고리즘과 규칙을 적용하는 머신러닝과 달리, 딥러닝에서는 심층 신경망을 통해 스스로 개, 고양이의 특성을 훈련하여 개와 고양이를 분류할 수 있습니다.
